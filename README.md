# blog

This repository houses the SurrealDB blog posts, in markdown format, which are published on the [SurrealDB website](https://surrealdb.com/blog), and on the [DEV community](https://dev.to/surrealdb). Blog posts are incrementally numbered in order of their publication. We welcome contributions and suggestions for potential blog posts. You can ask us a question on [Github discussions](https://github.com/surrealdb/surrealdb/discussions), or by tweeting [@tobiemh](https://twitter.com/tobiemh) or [@surrealdb](https://twitter.com/surrealdb) on Twitter. Alternatively chat with the [SurrealDB team on Discord](https://surrealdb.com/discord).
